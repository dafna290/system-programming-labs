Задание 1-2: Процессы fork/exec
Требования:
•Используйте: fork(), exec(), wait(), getpid(), getppid()
•Обрабатывайте ошибки
•Компиляция: gcc -Wall -Wextra -o task task.c

Вариант 6: Создать 4 дочерних процесса. Каждый i засыпает на i секунд (1,2,3,4).
После пробуждения выводит "Процесс i проснулся". Родитель ждёт всех.

Используемые системные вызовы:
fork() - создание нового процесса
getpid() - получение идентификатора текущего процесса
getppid() - получение идентификатора родительского процесса
sleep() - приостановка выполнения процесса
wait() - ожидание завершения дочернего процесса
exit() - завершение процесса

Дочерние процессы создаются почти одновременно
Каждый процесс засыпает на разное время
Процессы завершаются в порядке возрастания времени сна
Родительский процесс получает код завершения каждого дочернего процесса
Для запуска используется ./task


Пример вывода: 
Родительский процесс начался (PID: 2529, PPID: 305)
Дочерний процесс 1 создан (PID: 2530, PPID: 2529)
Процесс 1 засыпает на 1 секунд...
Дочерний процесс 2 создан (PID: 2531, PPID: 2529)
Процесс 2 засыпает на 2 секунд...
Дочерний процесс 3 создан (PID: 2532, PPID: 2529)
Процесс 3 засыпает на 3 секунд...

Родительский процесс (PID: 2529) ожидает завершения дочерних...
Дочерний процесс 4 создан (PID: 2533, PPID: 2529)
Процесс 4 засыпает на 4 секунд...
Процесс 1 проснулся
Дочерний процесс с PID 2530 завершился с кодом: 1
Процесс 2 проснулся
Дочерний процесс с PID 2531 завершился с кодом: 2
Процесс 3 проснулся
Дочерний процесс с PID 2532 завершился с кодом: 3
Процесс 4 проснулся
Дочерний процесс с PID 2533 завершился с кодом: 4
Все дочерние процессы завершены. Родительский процесс завершает работу.
