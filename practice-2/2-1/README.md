Задание 2-1: Многопоточность<br>
Требования:<br>
•Используйте библиотеку pthread<br>
•Компиляция: gcc -pthread -o task task.c<br>
•Функции: pthread_create(), pthread_join(), pthread_exit()<br>

Вариант 6: Создать 4 потока. Каждый вычисляет факториал числа (5,6,7,8). Результаты<br>
в глобальном массиве. Главный поток выводит все<br>

Используемые функции:<br>
pthread_create() - создание нового потока<br>
pthread_join() - ожидание завершения потока<br>
pthread_exit() - завершение потока<br>
Тип pthread_t - идентификатор потока<br>

Все 4 потока создаются почти одновременно<br>
Потоки выполняются параллельно<br>
Порядок завершения потоков непредсказуем<br>
Главный поток блокируется на pthread_join() до завершения всех дочерних потоков<br>

Для запуска используется ./task<br>

Пример вывода:<br>
4 потока:<br>
Поток 0: вычислил 5! = 120<br>
Поток 1: вычислил 6! = 720<br>
Поток 2: вычислил 7! = 5040<br>
Поток 3: вычислил 8! = 40320<br>

Результаты:<br>
Факториал 5! = 120<br>
Факториал 6! = 720<br>
Факториал 7! = 5040<br>
Факториал 8! = 40320<br>
